Timestamp Microservice

How to use this service:

Send a string as a URL parameter.

If the service recognizes the string as a date format or Unix timestamp, it will return both the natural language date (as month day, year) and the Unix timestamp (in seconds) in JSON format.

If the service does not recognize the string, it will return null for both values.

Examples

Input: https://sleepy-taiga-43144.herokuapp.com/12-26-2016

Output: {"unix":"1482710400","natural":"December 26, 2016"}


Input: https://sleepy-taiga-43144.herokuapp.com/1482710400

Output: {"unix":"1482710400","natural":"December 26, 2016"}


Input: https://sleepy-taiga-43144.herokuapp.com/lol-cats.html

Output: {"unix":null,"natural":null}
